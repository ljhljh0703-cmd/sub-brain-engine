# Sub-brain: The Intelligent Engine of Convergence (萬流歸宗)

> "글쓰기, 게임 개발, 그리고 AI—세 갈래의 강이 하나의 바다에서 만나는 지능형 엔진."

## 🧠 Philosophy: 만류귀종 (All Flows Return to the Source)
Sub-brain은 단순한 지식 저장소가 아닙니다. 소설 『[[초파리]]』의 서사적 깊이와 AI 엔지니어링의 기계적 로직, 그리고 게임의 상호작용성을 하나로 융합하여 **'살아있는 AI NPC'**를 창조하기 위한 지성적 모태입니다.

## 🏗️ Architecture: Dual-Engine System
- **Gemini CLI (Daily Ops)**: 지식의 수집(Ingest), 디벨롭, 그리고 실시간 오케스트레이션을 담당하는 기동 엔진.
- **Claude (SSOT)**: 지식의 무결성을 검증하고 최종 결정을 잠그는(Lock) 권위 엔진.
- **Graphify & PageIndex**: 파편화된 노트를 '의미의 망(Web of Meaning)'으로 연결하고, 추론 기반 트리 검색으로 대규모 맥락을 통제합니다.

## 🎨 Core Values
1. **탐미주의 (Aestheticism)**: 단순한 기능을 넘어, 지식 그 자체가 시각적/구조적 아름다움을 갖춘 '디지털 공예품'이 될 것을 지향합니다.
2. **강렬함 (Intensity)**: 미지근한 기록을 배제하고, 삶의 의의와 신념이 담긴 고밀도 지식만을 이식합니다.
3. **융합 (Convergence)**: 서로 다른 도메인(문학-기술-게임) 간의 'Surprising Connections'를 발굴하여 복리 지식을 창출합니다.

- **Ops**: Gemini CLI + Custom Skills (harness-factory, html-ppt-skill, etc.)
- **Knowledge**: Knowledge Graph (Graphify), Reasoning-based Indexing (PageIndex)
